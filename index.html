SIT223-SIT753 - Professional Practice in IT

Content

To be successful IT graduates need to understand the use of industry tools and practices, the ways these tools work and connect together, and the underlying professional, ethical, and teamwork knowledge and skills needed to put these into practice in a professional manner.

About the unit - To be updated

This unit aims to equip IT graduates with the essential blend of technical and professional skills for success in the modern IT industry. It covers industry-standard tools and practices like IT workflows, agile project management, DevOps pipelines, and version control, emphasizing how these elements integrate within companies. Crucially, the unit also addresses the necessary professional behaviors, teamwork abilities, and ethical considerations for working effectively. Furthermore, it tackles significant industry issues, particularly gender equity and diversity, to prepare future IT leaders to address these challenges. In essence, the unit provides both the technical know-how and the crucial soft skills and awareness needed for a professional and impactful IT career.
